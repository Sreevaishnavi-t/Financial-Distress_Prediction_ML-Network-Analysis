To run this Django project, please follow these steps:

1. Open your terminal or Windows PowerShell.
2. Navigate to the shortened project directory:
   cd "fd"
   
   (Note: The project was moved from its original location to this 'fd' named folder because Windows has a 260-character path limit, and the original deeply nested folder name was causing the application to fail to find 'settings.py').
   
3. Install the required dependencies using the Python launcher (py):
   py -m pip install django==4.2.19 scikit-learn imbalanced-learn matplotlib pandas seaborn

4. Run the development server:
   py manage.py runserver

5. Open your web browser and go to http://127.0.0.1:8000/ to view the application.
