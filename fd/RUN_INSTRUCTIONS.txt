To run this Django project, please follow these steps:

1. Open your terminal or command prompt.
2. Navigate to the project directory:
   cd "BPY_AIML_2519-A HYBRID NETWORK ANALYSIS AND MACHINE LEARNING MODEL FOR ENHANCED FINANCIAL DISTRESS PREDICTION"
3. Install the required dependencies:
   py -m pip install django==4.2.19 scikit-learn imbalanced-learn
4. Run the development server:
   py manage.py runserver
5. Open your web browser and go to http://127.0.0.1:8000/ to view the application.
